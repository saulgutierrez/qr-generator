# qr-generator
Simple web app for convert a link into a QR Code easy an quick. With this app you only need to give a valid link and click on Generate QR to create an scanneable image and share it. <br>
Technologies used:
<li>HTML</li>
<li>Bootstrap</li>
<li>Vanilla JS</li>
<li>QRCode.js</li>
<li>Google Fonts</li>
